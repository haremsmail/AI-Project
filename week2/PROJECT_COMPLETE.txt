📊 ALGORITHM CHARACTERISTICS
═══════════════════════════════════════════════════════════════════════════════

DEPTH-FIRST SEARCH (DFS)
├─ Pros: Simple, low memory for limited depth
├─ Cons: May not find shortest path, can be slow
└─ Best for: Small puzzle variants

BREADTH-FIRST SEARCH (BFS)
├─ Pros: Guarantees shortest path, explores systematically
├─ Cons: High memory usage
└─ Best for: Finding optimal number of moves

BEST-FIRST SEARCH (A*)
├─ Pros: Fastest, uses heuristics to guide search
├─ Cons: Requires good heuristic function
└─ Best for: Real-time applications, larger spaces
